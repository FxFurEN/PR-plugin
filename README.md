# PR-team-Plugin
<b> Plugin for rp server for Minecraft 1.19+</b><br>
The plugin allows you to play animations using the Emotecraft plugin.Interact with many minecraft functions and some plugins  <b>(MMOITems and MMOCore)</b>.

![preview](https://user-images.githubusercontent.com/114875125/232696460-d9701172-df06-4855-a45c-666c47c30416.gif)

In this example, the player is at a distance of <b>1.3</b> from the snow block, uses the SHIFT + RMB keyboard shortcut on the snow, an animation is produced, then after 60 ticks the snow disappears and the player gets a snowball
